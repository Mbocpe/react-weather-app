# Weather Forecast App
A simple React-based weather forecast application that displays daily weather summaries and detailed conditions such as temperature, humidity, wind speed, and pressure.

## Features
- 7-day weather forecast
- Daily summary with weather icon and description
- Expandable daily details (pressure, humidity, clouds, wind, etc.)
- Responsive, clean UI using CSS Grid and Flexbox

## Tech Stack
- React
- JavaScript (ES6+)
- CSS (Grid & Flexbox)
- OpenWeather API (or similar weather API)

## Getting Started

1. Install dependencies:
   ```bash
   npm install

2. Start the development server:
    ```bash
    npm start

3. Open your browser at:
    http://localhost:3000

## Notes
- Weather icons are loaded from a local icons/ directory.
- Forecast data is limited to the first 7 entries for display.
- This project focuses on UI layout and basic React state handling.
